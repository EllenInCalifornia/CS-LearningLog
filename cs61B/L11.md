# Interface Summary 
* In the last three lectures weâ€™ve seen how implements and extends can be used to enable **interface inheritance** and **implementation inheritance**.
* * Interface inheritance: What (the class can do).
* * Implementation inheritance: How (the class does it).
* All methods declared in an interface must be public
* Interface can have variables, but they must be public static final 
* A class can implement multiple interfaces.
